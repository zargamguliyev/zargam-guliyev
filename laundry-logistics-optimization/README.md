# Автоматизация логистики прачечной для сети отелей Oasis Retreats

## Обзор проекта

Данный проект посвящён анализу и улучшению логистического процесса между отелями и централизованным прачечным комплексом сети отелей **Oasis Retreats**.

В результате быстрого расширения сеть открыла **7 отелей в регионе за два года**, что значительно увеличило операционную нагрузку. Существующий логистический процесс перестал справляться с растущим объёмом перевозок постельного белья между отелями и центральной прачечной.

Цель проекта — проанализировать текущий логистический процесс и спроектировать систему, которая автоматизирует планирование доставок и повысит прозрачность движения постельного белья.

---

# Бизнес-проблема

Интервью с сотрудниками и внутренние обсуждения выявили ряд операционных проблем:

- доставки постельного белья планируются **без учёта загруженности отелей**
- в отелях **ограниченные складские площади**
- периодически возникает **дефицит постельного белья**
- водители следуют **неоптимизированным маршрутам**
- время прибытия доставок **непредсказуемо**
- расхождения в количестве белья выявляются **только при ежемесячной инвентаризации**
- утерянные позиции сложно отследить

Эти проблемы приводят к:

- неэффективному планированию логистики
- операционным сбоям в работе отелей
- финансовым потерям из-за утери белья

---

# Решение

Предлагаемое решение предусматривает внедрение **системы управления логистикой «L-Logistics»**.

Ключевые элементы решения:

- автоматизированное планирование доставок постельного белья  
- интеграция с **системой управления отелями «O! Hotel»**  
- планирование маршрутов для водителей  
- отслеживание движения белья между отелями и прачечной  
- повышение прозрачности управления запасами

Система использует данные о загруженности отелей для прогнозирования потребности в белье и улучшения планирования доставок.

# Моделирование системы

## Моделирование Use Case

Диаграммы Use Case разработаны для описания взаимодействия пользователей системы с логистической системой.

Основные участники:

- менеджер по бельевому хозяйству
- менеджер по логистике
- водитель

Ключевые сценарии использования:

- планирование доставок постельного белья
- составление расписания перевозок
- отслеживание движения белья
- управление задачами по доставке

---

## UML-моделирование

Для описания структуры системы были разработаны следующие UML-диаграммы:

**Диаграмма классов**

Диаграмма классов определяет основные сущности системы:

- отель
- запасы постельного белья
- заявка на доставку
- маршрут доставки
- задача на перевозку

**Диаграмма объектов**

Диаграмма объектов демонстрирует взаимодействие сущностей системы в процессе планирования доставки постельного белья.
